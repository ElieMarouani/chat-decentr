# Dépôt chat-decentr
Author      Eli Marouani, Tom De Rudder  
Version     0.0.1

> "_chat-decentr_" est un chat décentraliser.

## Architecture
```bash
	chat-decentr
	├── client
    │   ├── client.cpp
    │   └── makefile
    ├── server
    │   ├── makefile
    │   └── server.cpp
	└── README.md
```

## Pré requis
- g++
- zmq
- git

## Installation
```bash
    git clone "https://github.com/ElieMarouani/chat-decentr.git" chat-decentr
    cd ./chat-decentr
```
